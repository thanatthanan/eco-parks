# Project Instructions (อุทยานแห่งชาติ)

## Workflows
- **Synchronized Updates:** ทุกครั้งที่มีการอัปเดตข้อมูลการเดินทางใน `CHECKLIST.md` หรือ `TRAVEL_DASHBOARD.md` จะต้องอัปเดตข้อมูลในไฟล์ต่อไปนี้ให้สอดคล้องกันเสมอ:
    1.  `TRAVEL_MAP.html` (แผนที่และหมุดอุทยาน)
    2.  `TRAVEL_DASHBOARD.html` (หน้าแดชบอร์ดสรุปสถิติและกราฟความคืบหน้า)
- **Date Format:** ใช้ปีพุทธศักราช (พ.ศ.) ในไฟล์ Markdown และใช้ชื่อจังหวัดภาษาอังกฤษในโครงสร้างข้อมูลของไฟล์ HTML แผนที่ เพื่อให้รองรับกับ GeoJSON
